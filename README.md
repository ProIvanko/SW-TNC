## Об игре

Вы пилотируете TIE Defender на стороне Империи в сражении против флота Повстанцев.  
В бою участвуют две эскадрильи истребителей и два флагмана — Звёздный Разрушитель против крейсера Mon Calamari.

Управление:
- `W A S D Q E` + мышь — пространственный полёт (FlyControls, 6 степеней свободы)
- `ЛКМ` — лазерный залп
- Поддержка геймпада

## Стек

| | |
|---|---|
| Рендеринг | Three.js (WebGL), GLTFLoader, CSS2DRenderer |
| Логика игры | Vanilla JS (ES Modules), без фреймворков |
| Авторизация | Firebase Authentication |
| 3D-модели | GLTF-файлы (TIE Defender, A-Wing, X-Wing, Y-Wing, TIE Fighter, Star Destroyer, Mon Calamari, Tatooine) |
| Аудио | Web Audio API (лазеры, марш Империи) |

---

## Что реализовано

- **AI противников** — 20 истребителей Повстанцев патрулируют сектор и атакуют игрока
- **Система стрельбы** — лазерные болты с коллизией, перегрев орудия при частой стрельбе
- **Два флагмана** — статические крупные корабли задают масштаб сражения
- **Окружение** — объёмный звёздный фон (Milky Way GLTF) и планета Татуин
- **CSS2DRenderer** — метки над объектами поверх 3D-сцены
- **Авторизация** — вход по логину/паролю через Firebase; данные пользователя хранятся в облаке
- **Настройки** — регулировка чувствительности, громкости, качества графики

## Архитектура

```
src/
├── main.js               — вход, авторизация, навигация между экранами
├── battleModule.js       — инициализация сцены, рендер-луп, управление игроком
├── starfightersInit.js   — AI противников, их движение и стрельба
├── environmentInit.js    — загрузка окружения (фон, планета, освещение)
├── battleModule.js       — коллизии лазеров, подсчёт очков
├── audioManager.js       — пул аудио-объектов, проигрывание звуков
├── gamePadModule.js      — поддержка Gamepad API
├── userDataStorage.js    — Firebase-интеграция, профиль игрока
└── userInterFace.js      — HUD (очки, здоровье, перегрев орудия)
```

## Контекст проекта

Финальный проект курса **«Разработка WEB-приложений с использованием JavaScript»** (IT-Academy, 2023).  
Цель — освоить Three.js на практике через задачу с понятной механикой: физика движения, игровой AI, сцена с большим числом объектов.
